DTC-TIME은
특정 회차의 시작 시간을 기준으로
앞으로 이어질 방송 시간표를 자동으로 계산해주는 프로그램입니다.

설치 방법

ZIP 파일을 다운로드합니다.
바탕화면 또는 원하는 위치에 새 폴더를 하나 만듭니다.
ZIP 파일을 반드시 새 폴더 안에 압축 해제합니다.
압축이 완료되면 폴더 안에 있는 DTC-TIME.html 파일을 더블클릭하여 실행합니다.

※ ZIP 파일을 바로 열어서 실행하지 마십시오.
반드시 압축을 푼 후 사용해야 합니다.


 사용 방법
① 기준 시간 입력

상단의 입력란에 다음 정보를 입력합니다.
연 월 일 시 분 기준 에피소드
이 값은 “해당 회차가 실제로 시작한 시간”입니다.
입력한 내용은 자동 저장됩니다.
입력시간은 24시간제로 입력해야 합니다.

기준 시간을 모를때에는 방송중인 채팅에 질문 하시면
다른 분들이 시간표를 보여줄 수 있으니 그 시간을 참고하시면 됩니다.

② 시즌 선택

시즌1~시즌4 중 원하는 회차 버튼을 클릭합니다.
클릭 즉시:
해당 회차부터 6개 일정이 자동 계산되어 표시됩니다.
다음 방송일이 하단에 표시됩니다.
현재 진행 중 회차가 자동 강조 표시됩니다.

③ 결과 확인

출력 예시:
11:12-시즌3 스페셜 / 12:31-백투더 경성 / ...
각 회차의 시작 시각만 표시됩니다.
총 6개 회차가 자동 생성됩니다.

④ 복사 기능

일정 복사 버튼 → 위 시간표 복사
다음 방송일 복사 버튼 → 다음 시작일 복사

유의사항

기준 시간은 정확히 입력해야 합니다.
날짜나 시간이 틀리면 전체 계산이 달라집니다.
